## 常用命令

```bash
sudo docker login --username=username registry.cn-shanghai.aliyuncs.com


docker push registry.cn-shanghai.aliyuncs.com/wangrollin-web/web-front:0.0.1

docker pull registry.cn-shanghai.aliyuncs.com/wangrollin-web/web-front:0.0.1

docker run -d -p 80:80 registry.cn-shanghai.aliyuncs.com/wangrollin-web/web-front:0.0.1




docker push registry.cn-shanghai.aliyuncs.com/wangrollin-web/rancher-server:stable



docker pull rancher/server:stable

docker tag rancher/server:stable registry-internal.cn-shanghai.aliyuncs.com/wangrollin-web/rancher-server:stable

docker push registry-internal.cn-shanghai.aliyuncs.com/wangrollin-web/rancher-server:stable
```
